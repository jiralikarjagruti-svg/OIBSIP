# AuthVault — Login Authentication System
## Jagruti Jiralikar | OIBSIP Web Development Internship | Level 2 — Task 4

A premium, secure client-side authentication system with Registration, Login, and a protected Dashboard.

## Tech Stack
HTML5 | CSS3 | JavaScript (Vanilla) | Web Crypto API (SHA-256)

## Features
- User Registration with full validation
- Duplicate email detection
- Password strength indicator (Weak / Fair / Good / Strong / Very Strong)
- SHA-256 password hashing — passwords never stored as plain text
- Login with incorrect credential handling
- Protected Dashboard — only accessible after login
- Session management using sessionStorage
- Auto session restore on page refresh
- Secure Logout — clears session completely
- Live session timer on dashboard
- Account details panel (name, email, joined date, last login)
- Security status panel showing protection features
- Loading spinner animations
- Toast notifications
- Particle canvas background
- Fully responsive (mobile + desktop)

## How to View
Open index.html in any browser.
Register a new account → Login → Access protected dashboard.
