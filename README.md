# Nikola Tesla — A Tribute | Jagruti Jiralikar
## OIBSIP Web Development Internship | Level 2 — Task 2

A modern and responsive tribute page dedicated to Nikola Tesla, showcasing his life, achievements, inventions, and lasting legacy through an elegant user interface.

## Tech Stack
HTML5 | CSS3 | JavaScript (Vanilla)

## Features
- Modern dark-themed responsive design
- Hero section with animated visuals
- Biography section with key highlights
- Interactive timeline of Nikola Tesla's life
- Achievement cards showcasing major inventions
- Inspirational quote section
- Legacy section with elegant typography
- Smooth scrolling and subtle animations
- Mobile-friendly responsive layout

## Sections Included
- Hero Section
- Biography
- Inspirational Quote
- Timeline
- Major Inventions
- Legacy

## How to View
Open `tribute-page.html` in any modern web browser.
