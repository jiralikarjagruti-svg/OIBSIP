# Calculator — Jagruti Jiralikar
## OIBSIP Web Development Internship | Level 2 — Task 1

A fully functional browser-based calculator with animations.

## Tech Stack
HTML5 | CSS3 | JavaScript (Vanilla)

## Features
- Basic arithmetic (+, −, ×, ÷)
- Clear, Backspace & Decimal support
- Division by zero error handling
- Operator chaining
- Ripple effect animations
- Keyboard support
- CSS Grid layout

## How to View
Open index.html in any browser.
