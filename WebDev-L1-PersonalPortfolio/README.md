# Personal Portfolio — Jagruti Jiralikar
A personal portfolio website showcasing my skills, projects, and certifications. Built as part of the OIBSIP Web Development Internship (Level 1 - Task 2).

## Tech Stack
HTML5, CSS3, JavaScript

## Features
- Animated hero section with code panel
- Skills section (categorized by language/tool)
- Projects section with GitHub links
- Certifications section (Cisco Networking Academy)
- Fully responsive

## How to View
Open index.html in any browser.
