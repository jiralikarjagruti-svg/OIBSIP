# Landing Page — Swaply
A visually polished static landing page built as part of the OIBSIP Web Development Internship (Level 1 - Task 1).

## Tech Stack
HTML5, CSS3, JavaScript

## Features
- Sticky navigation bar
- Hero section with CTA button
- Features and Testimonial sections
- Fully responsive (mobile-friendly)
- Smooth scroll navigation

## How to View
Open index.html in any browser.
