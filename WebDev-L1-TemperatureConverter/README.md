# Temperature Converter — Jagruti Jiralikar
An interactive temperature converter with a live animated thermometer. Built as part of the OIBSIP Web Development Internship (Level 1 - Task 3).

## Tech Stack
HTML5, CSS3, JavaScript (Vanilla)

## Features
- Converts between Celsius, Fahrenheit, and Kelvin
- Live animated thermometer visual
- Color changes based on temperature
- Input validation and absolute zero error handling
- Fully responsive

## How to View
Open index.html in any browser.
