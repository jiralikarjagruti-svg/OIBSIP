# TaskFlow — To-Do Web App
## Jagruti Jiralikar | OIBSIP Web Development Internship | Level 2 — Task 3

A premium, fully interactive To-Do web application built with vanilla HTML, CSS, and JavaScript.

## Tech Stack
HTML5 | CSS3 | JavaScript (Vanilla)

## Features
- Add, Edit, and Delete tasks
- Mark tasks as Complete/Pending
- Priority system — Low, Medium, High (color coded)
- Filter tasks — All, Pending, Completed, High Priority
- Live progress bar showing completion percentage
- Task count stats (Total, Pending, Done, High Priority)
- Timestamps on every task (added + completed time)
- Tasks persist across page refreshes using localStorage
- Toast notifications for all actions
- Smooth animations and glassmorphism UI
- Particle canvas background
- Fully responsive (mobile + desktop)

## How to View
Open index.html in any browser — no setup needed.
